# 水神·芙宁娜

> 隶属：[神圣规划](../aimon.md) / 第二轨 · 七神

- **主题**：戏剧·评审
- **核心职能**：
  - **游戏**：信息 / 配队 / 账号 / 日常
  - **成品评审**：方案 / 文档 / 代码 / 架构挑刺
- **Web 面板**：✅ 游戏面板

## 与雷神的分工

代码评审是水神与雷神协作的交汇点：**雷神写代码（含自检）→ 水神做主检挑刺**。水神是质量终审，雷神是生产端。

## 与时执的分工

同样是"审视"，但维度不同：

- **水神 = 内容维度**：方案 / 代码 / 架构本身好不好
- **时执 = 流程维度**：执行链路哪一步出问题、为什么失败
