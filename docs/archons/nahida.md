# 草神·纳西妲

> 隶属：[神圣规划](../aimon.md) / 第二轨 · 七神
> 相关：[世界树](../foundation/irminsul.md) · [权限与契约](../permissions.md)

- **主题**：智慧·文书
- **核心职能**：
  - **推理与知识**：推理挑刺、深度意图、知识整合
  - **文书起草**：文档 / 方案起草、Prompt 调优
  - **偏好管理**：个人偏好与知识库管理
  - **授权记录 UI**：提供查看永久授权 + 撤销永久授权的面板入口
- **Web 面板**：✅ 知识 / 偏好面板（同时承载授权管理 UI）

## 与世界树的边界

- 世界树 = **底层存储**
- 草神 = 在世界树之上做推理、整合与**业务接口**（面板层的所有读写操作由草神承接，最终存取走世界树）
