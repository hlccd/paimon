# 关键边界对照表

> 隶属：[神圣规划](aimon.md)

汇总所有模块的关键职责归属，避免交叉混淆。按场景分类便于速查。

## 1. 入口 & 意图

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| 意图识别（粗） | 派蒙 | "走哪条路" |
| 意图推理（深） | 草神 | "用户真正想要什么" |
| 闲聊响应 | 派蒙 | 浅层 LLM，不经天使 |

## 2. 安全与审查

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| 安全拦截（轻） | 派蒙 | 关键词 / 格式 |
| 安全审查（深） | 死执 | LLM 审核 + 越权 + 合规 |
| 内容评审 | 水神 | 方案 / 代码 / 文档 / 架构挑刺 |
| 流程审计 | 时执 | 执行链路复盘 |

## 3. 编排 & 执行

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| 任务编排 | 生执 | DAG 拆分、轮次控制 |
| 子任务路由 | 空执 | 路由到七神 |
| 写代码 | 雷神 | 生成、重构 |
| 执行代码 | 火神 | shell / runtime |

## 4. 时间 & 归档

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| 运行时压缩 | 时执 | 活跃会话 token 压缩 |
| 最终归档 | 时执 | 结束后入博物馆 |

## 5. 资源计量

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| Token + 花费统计 | 原石 | 按模块 / 用途 / 会话 多维度聚合 |
| 理财业务 | 岩神 | 用户个人财富（股票 / 资产 / 退休） |

## 6. Skill 生态

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| skill 生态管理 | 冰神 | 发现 + 注册世界树 + AI 自举 |
| skill 写入 | 冰神 | 启动扫入 / 运行时审查通过后写世界树（唯一写入者） |

## 7. 权限画像与授权

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| 权限画像（权威存储） | 世界树 | 涉及权限的两个数据域：skill 声明 + 用户授权记录（世界树总共承载 9 个域） |
| 权限画像本地缓存 | 派蒙 / 死执 | 启动读世界树；运行时由四影通知更新 |
| 权限询问（天使路径） | 派蒙 | 单项询问 + 永久关键词识别 + 写世界树 |
| 权限确认（复杂任务） | 死执 | 扫描 DAG + 批量一次性确认 |
| 用户永久授权写入（对话） | 派蒙 | 识别"永久"关键词 → 直接写世界树 + 自更新缓存（不跨模块） |
| 用户永久授权查看（面板） | 草神面板 | UI 展示时读取世界树 |
| 用户永久授权撤销（面板） | 草神面板 | UI 撤销入口，触发对世界树的写入 |

## 8. 推送链路

| 场景 | 归属 | 关键区分点 |
|---|---|---|
| 推送内容整理 | 数据收集者（风神 / 岩神 等） | 谁的数据谁整理 |
| 推送响铃（触发） | 三月 | 定时响铃 + 接收事件响铃请求 |
| 推送送达 | 派蒙 | 唯一出入口，人格包装 |
