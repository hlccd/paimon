# 空执·阿斯莫代

> 隶属：[神圣规划](../aimon.md) / 第二轨 · 四影
> 整体调用顺序：**死执 → 生执 → 空执 → 七神 → 时执**

**定位**：原初四影之空执，执掌空间与移动，对应任务路由。

## 核心能力

- **子任务路由**：自动路由到对应七神能力模块
- **服务发现**：新模块自动接入，无需改路由规则
- **故障切换**：七神业务实例故障时自动切换到健康实例（保证任务不中断）
- **多子任务并发**：本层天然实现（多个子任务同时派发）

## 与神之眼的边界

空执管**七神业务实例**故障切换（七神模块级）；**LLM 底层实例**故障由神之眼自处。
